# CodeReapers

## Starting Project

1. cd backend
2. npm install
3. cd frontend
4. npm install
5. npm run build

### to run both

6. pm2 start ecosystem.config.js

## How to start frontend App

1. npm install
2. npm run build
   Using node only
3. npm run start
   Using PM2
4. pm2 start npm --name "nextjs-app" -- start
